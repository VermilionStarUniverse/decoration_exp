# 概念梳理
## 主材辅材
### 辅材: 外观看不见的材料
- 水管，电线，水泥，砂浆
### 主材: 看得见, 直接影响效果的材料
- 瓷砖，地板，洁具，卫具

## 全包半包清包
- 全包: 施工方提供主材辅材
- 半包: 施工方提供主材, 辅材要自理
- 清包: 施工方不提供材料, 主材辅材都要自理

## 硬装软装
推荐减少硬装, 使用软装搭配出心仪的效果.
(把房屋倒置, 会掉下来的是软装, 不会掉下来的是硬装)
### 硬装: 不能移动的
- 水电，墙面，地面，洁具，卫具
### 软装: 方便移动替换的
- 沙发，茶几，窗帘，布艺，摆件，绿植
