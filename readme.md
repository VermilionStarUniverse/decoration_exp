# 概念梳理
## 主材辅材
### 辅材: 外观看不见的材料
- 水管，电线，水泥，砂浆
### 主材: 看得见, 直接影响效果的材料
- 瓷砖，地板，洁具，卫具

## 全包半包清包
- 全包: 施工方提供主材辅材
- 半包: 施工方提供主材, 辅材要自理
- 清包: 施工方不提供材料, 主材辅材都要自理

## 硬装软装
推荐减少硬装, 使用软装搭配出心仪的效果.
(把房屋倒置, 会掉下来的是软装, 不会掉下来的是硬装)
### 硬装: 不能移动的
- 水电，墙面，地面，洁具，卫具
### 软装: 方便移动替换的
- 沙发，茶几，窗帘，布艺，摆件，绿植

# 预算规划三大原则
## 0 安全项目，诸如：阳台防护、天然气报警、防漏电等务必排除所有隐患
## 1 性价比
- 额外的花费是否带来急需的性能？
## 2 体验
- 体验的提示能不能明确被感知到？
- 这种体验是不是很必要？
- 如：一千与一万的床难有体验提升，但一千与一万的床垫更有可能带来更好的体验。
## 3 后期成本

# 大小品牌选择
## 有技术壁垒的选择大品牌
- 如家电，因研发实力和专利保护，大厂能做，小厂做不了。
## 技术要求不高，或产业很成熟的，选小厂
- 如全屋定制，瓷砖
## 同等条件下一定选大品牌
- 不信小品牌的差异化营销
## 不用高端品牌
## 低端品牌高端产品 > 高端品牌低端产品
## 线上线下？
- 中央空调、地暖、窗等高度依赖本地化服务选线下
- 所有的标品，不管主材、辅材、家具、成品家具可考虑线上
## 智能家居
- 行业仍不成熟，简单弄弄尝尝鲜
- 成熟的智能设备：智能锁、智能马桶、扫地机器人
- 智障设备：~~智能冰箱~~、~~智能垃圾桶~~、~~智慧屏~~
- 智能生态：智能开关、智能网关可体验，再复杂不行。
